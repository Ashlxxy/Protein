===============================================================
              TUGAS PROYEK TEKNOLOGI INFORMASI
===============================================================

Project ini merupakan prototype website fashion modern
yang dikembangkan dengan fokus pada tampilan, struktur halaman,
dan pengalaman pengguna (UI/UX).

Website ini dapat dikembangkan lebih lanjut menjadi sistem
e-commerce lengkap yang terhubung dengan database atau API.

---------------------------------------------------------------
                 ✦   PROJECT HIGHLIGHTS   ✦
---------------------------------------------------------------

- Tampilan modern dan minimalis
- Responsif untuk mobile, tablet, dan desktop
- Masonry gallery untuk halaman Community
- Sistem halaman modular mudah dikembangkan
- Placeholder konten untuk blog dan about section

---------------------------------------------------------------
