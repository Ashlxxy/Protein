## 🏷️ **Project Overview**

Website ini dirancang sebagai prototipe e-commerce fashion modern yang dapat dikembangkan menjadi sistem lengkap.

✨ Dibuat dengan fokus pada:

| Fokus | Status | Detail |
|-------|--------|--------|
| 🎨 UI/UX Modern | ✔ | Desain minimalis, clean, responsive |
| 📱 Mobile Friendly | ✔ | Layout adaptif untuk tablet & smartphone |
| 📦 Modular Pages | ✔ | Struktur HTML terpisah & maintainable |
| 🔧 Expandable System | ✔ | Bisa dikembangkan ke API / database |

## 🚀 Cara Menjalankan

1. **Download ZIP**
2. Extract ke folder lokal
3. Buka `index.html` di browser  
   **(Chrome, Firefox, Edge recommended)**

> 📌 Tidak memerlukan database atau server — _fully static prototype._
