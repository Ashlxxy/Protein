Buitenworks demo v2 - Frontend package
Files:
- index.html, catalog.html, account.html, login.html
- styles.css, scripts.js
- assets/ (logo + product placeholders)

How to run:
1. Extract ZIP to folder, open index.html in browser, or run a small http server:
   python -m http.server 8000
   open http://localhost:8000/

Replace assets/buitenworks-logo.png with your real logo to match visual exactly.
